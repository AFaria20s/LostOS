# 0x194OS

A bare-metal x86 operating system written in C.
0x194 = 0x194. do the math ;)

---

## Dependencies

```
sudo apt install gcc nasm qemu-system-x86 grub-pc-bin xorriso make
```

## Running

```
git clone https://github.com/AFaria20s/0x194OS
cd 0x194OS
make run
```

## Build .iso only

```
make
```
